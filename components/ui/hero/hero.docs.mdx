## Hero Component

A hero component that renders an image, a message and some cta links

### Using props to customize the text

Modify the text to see it change live:

```js live
<Hero
  src="https://images.unsplash.com/photo-1542831371-29b0f74f9713?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=2250&q=80"
  alt="bit"
  text="Bit is a free and distributed version control system for components and dependencies. It helps build component driven apps in a simple and scalable way."
  headingText="bit is git for Components"
  secondaryLinkText="Watch Tutorial"
  mainLinkText="Go to the Docs"
  mainLinkHref="/"
  secondaryLinkHref="/"
/>
```
